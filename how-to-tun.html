How to Run Your Portfolio Website

This guide will help you set up and run your portfolio website, including how to use the script files.

Project Structure

portfolio/
├── index.html          # Main HTML file
├── styles.css          # CSS styles
├── script.js           # Main JavaScript functionality
└── scripts/
    ├── email-service.js    # Email configuration helper
    └── animations.js       # Animation configuration helper

Step 1: Basic Setup

First, make sure all files are in the correct folder structure as shown above.

Step 2: Running the Website

You have several options to run the website:

Option 1: Open directly in browser

Simply double-click the index.html file to open it in your default web browser.

Option 2: Use a local server (recommended)

For the best experience, use a local development server:

If you have Node.js installed:

# Install a simple HTTP server
npm install -g http-server

# Navigate to your project folder
cd path/to/portfolio

# Start the server
http-server

If you have Python installed:

# Navigate to your project folder
cd path/to/portfolio

# Python 3
python -m http.server

# Python 2
python -m SimpleHTTPServer

Then open your browser and go to http://localhost:8080 or the URL shown in your terminal.

Step 3: Running the Helper Scripts

The helper scripts (email-service.js and animations.js) are meant to provide guidance for configuring your portfolio. You can run them in several ways:

Option 1: Using the browser console

1. Open your portfolio website in the browser

2. Open the browser's developer tools (F12 or right-click > Inspect)

3. Go to the Console tab

4. Copy and paste the content of the script you want to run

5. Press Enter to execute

Option 2: Using Node.js

# Navigate to your scripts folder
cd path/to/portfolio/scripts

# Run the email service script
node email-service.js

# Run the animations script
node animations.js

Step 4: Configuring Email Service

After running the email-service.js script, follow the instructions to set up EmailJS:

  1. Sign up for EmailJS at https://www.emailjs.com/
  2. Create a service connecting to your email provider
  3. Create an email template with the variables mentioned
  4. Get your User ID, Service ID, and Template ID
  5. Update the script.js file with your credentials

Step 5: Customizing Animations

After running the animations.js script, you'll get instructions on how to customize the animations in your portfolio:

  1. Modify particle animations in the ParticleAnimation class in script.js
  2. Adjust scroll animations in the ScrollAnimations class
  3. Add additional animation classes to your CSS
  4. Apply these animation classes to elements in your HTML

Note: The helper scripts are primarily educational and provide guidance on how to configure your portfolio. They don't make actual changes to your files - you'll need to implement their suggestions manually.

Customizing Your Portfolio

To personalize your portfolio:

  • Replace "Your Name" with your actual name throughout the HTML
  • Update the profile picture and project images
  • Modify the skills and their percentages
  • Add your own projects with descriptions and links
  • Update contact information with your actual details
  • Customize colors and styles in the CSS file
